# **MemeZilla** by YASHWANTH M

**MemeZilla** is a fun little React app that fetches fresh memes from the Meme API and serves them straight to your screen. No effort required — just press one button.

The app is live now!  
Check out **MemeZilla** by clicking [here](https://yashlist.vercel.app/).

![App Preview](./src/assets/app_preview.png)

## Tech Stack

- ⚛️ **React.js** — The framework behind my app
- 🎨 **Tailwind CSS** — Used for styling
- 😎 **Meme API** — The main character

## What I learned from this project

Working on this project gave me hands-on experience and taught me this:

- Mobile-first design is best for achieving responsiveness.
- Using `useRef` for creating handle to grab elements.
- Using `useState` for the meme URL.
- Fetching data from the meme API.
- **Component thinking** - breaking UI into reusable pieces.
